# 在线图书商城管理系统
## 项目介绍
本项目是一款B2C在线图书电商平台，分为普通用户购书前端 + 商家管理后台双端口，实现完整的线上图书购买、店铺运维全业务流程。

## 技术栈
后端：SpringBoot
数据库：MySQL 8.0
前端：Vue3 + Element Plus
部署：Tomcat 9.0

## 核心功能
### 用户端
账号注册登录、图书分类浏览、关键词搜索、购物车管理、下单结算、个人中心、订单全程追踪

### 管理后台
管理员专属权限登录、图书上下架维护、全站订单管理、用户账号管控、首页公告运营

## 项目亮点
1. 双角色权限体系，精细化访问控制
2. 数据库结构规范，可灵活扩展迭代
3. 页面交互流畅，UI美观适配
4. 全程独立完成开发、调试、部署全流程
