# 图书分类与阅读指南

## 文学小说

### 经典文学
经典文学作品跨越时代，探讨人性、社会与命运的永恒主题。推荐入门作品：《百年孤独》（马尔克斯，魔幻现实主义巅峰）、《1984》（奥威尔，反乌托邦经典）、《红楼梦》（曹雪芹，中国古典文学巅峰）、《活着》（余华，中国当代文学代表作）。

### 科幻小说
科幻小说探索技术与人性的交汇。入门推荐：《三体》（刘慈欣，中国科幻里程碑）、《银河帝国》系列（阿西莫夫）、《沙丘》（赫伯特）、《神经漫游者》（吉布森，赛博朋克开山之作）。硬科幻适合理工科背景的读者，软科幻更注重社会与人性探讨。

### 推理悬疑
推理小说以解谜为核心乐趣。经典推荐：阿加莎·克里斯蒂《无人生还》、东野圭吾《白夜行》和《嫌疑人X的献身》、柯南·道尔《福尔摩斯探案集》。日本推理注重动机分析，欧美推理注重逻辑推理。

## 科技与编程

### 编程入门
编程入门建议从 Python 或 JavaScript 开始，语法简洁反馈快速。推荐书籍：《Python编程：从入门到实践》、《JavaScript高级程序设计》、《Head First设计模式》。

### Java 进阶
Java 开发者必读：《深入理解Java虚拟机》（周志明，JVM原理与调优）、《Effective Java》（Joshua Bloch，最佳实践）、《Java并发编程实战》（Brian Goetz，多线程编程）。

### 系统架构
架构设计经典：《系统设计》、《微服务架构设计模式》、《凤凰架构》（周志明）、《领域驱动设计》（Eric Evans，DDD奠基之作）。

## 商业与经济

### 经济学入门
推荐：《经济学原理》（曼昆，经典教材）、《薛兆丰经济学讲义》（通俗易懂）、《国富论》（亚当·斯密，经济学开山之作）。

### 创业与管理
推荐：《从0到1》（彼得·蒂尔）、《精益创业》（Eric Ries）、《创新者的窘境》（克里斯坦森）。

## 人文社科

### 心理学
入门推荐：《心理学与生活》（津巴多）、《思考，快与慢》（丹尼尔·卡尼曼，行为经济学）、《社会心理学》（戴维·迈尔斯）。

### 历史
推荐：《人类简史》（尤瓦尔·赫拉利）、《万历十五年》（黄仁宇）、《明朝那些事儿》（当年明月，通俗历史典范）。

### 哲学
入门推荐：《苏菲的世界》（乔斯坦·贾德，哲学小说）、《西方哲学史》（罗素）、《中国哲学简史》（冯友兰）。

## 阅读建议

### 睡前阅读
适合睡前的轻松读物：散文集（如汪曾祺、林清玄作品）、短篇小说集、轻科普读物、诗歌。避免选择过于紧张刺激或需要深度思考的书籍。

### 通勤阅读
通勤时适合阅读：杂志文章、短篇非虚构、自我提升类书籍、有声书。篇幅短小、随时可以中断的书籍最佳。

### 深度学习
需要集中精力深度阅读的书籍：专业教材、哲学著作、技术书籍。建议安排整块时间，做好笔记。
